O que é musculação?

O treinamneto de força conhecido como musculação é um forma de exercício contra resistência.A msuculação tem como intuito, aumentar a sua massa corporal. Para isso são usados equipamentos para forçar os musculos, que geram contrações. O treino é praticado para fins estéticos, para melhorar a saúde ou para melhorar o desempenhos de atletas.
